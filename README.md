# Nuvoton_N76E003_LCDTest
Nuvoton N76E003 LCD Test Code support with 4bit and 8bit mode and Floating Point Display

<h3>PinMap</h3>

LCD Data Port<br>
DATA PORT P1<br>
D4 -- 4<br>
D5 -- 5<br>
D6 -- 6<br>
D7 -- 7<br>

CONTROL PORT P1<br>
Enable pin -- 3<br>
R/W pin    -- 2<br>
RS pin     -- 1<br>

DEBUG LED -- P10
